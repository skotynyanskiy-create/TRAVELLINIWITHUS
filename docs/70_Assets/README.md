# 70_Assets

Cartella allegati del vault Obsidian. Configurata come `attachmentFolderPath` in `docs/.obsidian/app.json`.

Ogni immagine, PDF, file caricato o incollato in una nota viene salvato qui automaticamente.

Non organizzare manualmente le sottocartelle: Obsidian gestisce il placement.
